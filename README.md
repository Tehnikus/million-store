# Million store
E-commerce store based on Laravel and PostgreSQL
Features
 - Multi tenancy: Create several separated sites that may (or may not) share products, categories, and every other entity - all with distinct prices, pictures, descriptions and all being controlled from one admin panel
 - Multi language: Every store can have different frontend languages, including all store content, such as descriptions and SEO tags
 - Multi currency: Set each product price in every currency separately
 - First class SEO: Crafted and experience based SEO tools to control your SEO content, improve ranking and make SEO easier
 - Facet filtering with landing SEO pages: Complex and fast facet navigation but with stable static URLs and the ability to set page descriptions
 - Full text fuzzy search with ranking: Thanks to PostgreSQL vectors store search engine will find everything from customer's search, even with typos
 - Full fledged JSON-LD markup: Most compregensive markup for rich snippets and generated AI search
